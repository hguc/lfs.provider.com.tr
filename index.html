Sıfırdan Linux (Linux From Scratch)' a Hoşgeldiniz!

Sıfırdan Linux (LFS), tamamen kaynak kodundan kendi özel Linux çekirdekli işletim sisteminizi oluşturmak için adım adım talimatlar sağlayan bir projedir.

Currently, the Linux From Scratch organization consists of the following subprojects:

  • LFS :: LFS Linux From Scratch yani sıfırdan Linux baş harfleri ile türetilmiştir. LFS, diğer tüm projelerin türetildiği temel olan ana kitaptır.
  • BLFS :: Beyond Linux From Scratch helps you extend your finished LFS installation into a more customized and usable system.
  • ALFS :: Automated Linux From Scratch provides tools for automating and managing LFS and BLFS builds.
  • İpuçları :: The Hints project is a collection of documents that explain how to enhance your LFS system in ways that are not included in the LFS or BLFS books.
  • Yamalar :: The Patches project serves as a central repository for all patches useful to an LFS user.
  • LFS Editör Kılavuzu :: LFS geliştirme sürecini açıklayana doküman.
